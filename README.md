# Prem Shinde Portfolio (HTML/CSS/JS)

Premium responsive personal portfolio website built using plain HTML, CSS, and JavaScript.

## Features

- Dark futuristic UI with glassmorphism cards
- Responsive layout for desktop, tablet, and mobile
- Sticky navbar with active section highlight
- Typing effect, reveal animations, skill bars, counters
- Project, experience, certificates, achievements, GitHub, and contact sections
- Scroll progress bar, custom cursor glow, floating social dock
- Loader screen, certificate modal, and back-to-top button
- SEO metadata + Open Graph tags

## Tech

- HTML5
- CSS3
- Vanilla JavaScript (ES6)

## Run Locally

You can open `index.html` directly in your browser, or run a local static server:

```bash
npx serve .
```

## Deployment

### Netlify

1. Connect repo to Netlify
2. Build command: `echo 'Static site, no build step'`
3. Publish directory: `.`

Or use the included `netlify.toml`.

## File Structure

- `index.html` - main single-page portfolio
- `style.css` - all styles and responsive rules
- `script.js` - dynamic rendering and animations
- `public/` - static assets (favicon, OG image)
- `src/assests/Prem_Gokul_Shinde_Resume.pdf` - resume download file
